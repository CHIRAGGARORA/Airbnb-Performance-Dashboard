# Global Airbnb Performance Dashboard

An interactive Power BI dashboard analyzing Airbnb business performance, customer behavior, market trends, trust metrics, and seasonality patterns across major global cities.

---

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

---

## Key Insights

- Nearly 99% of users posted 3 reviews or fewer.
- Paris, New York, and Sydney account for almost half of total listings.
- Mexico City and Rio de Janeiro received the highest overall ratings.
- Over two-thirds of hosts are fully verified.
- Review activity reflects strong seasonal travel behavior.
